# NotFound

The NotFound component is used to display a "not found" or "access denied" page.

## Props

| Prop        | Type     | Default | Description                              |
|-------------|----------|---------|------------------------------------------|
| statusCode  | number   | 404     | Status code to display                   |
| title       | string   | ""      | Page title                               |
| description | string   | ""      | Additional description                   |

## Usage Example

  <Story name="Default">

</Preview>

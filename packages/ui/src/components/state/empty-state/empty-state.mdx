# EmptyState

The EmptyState component is used to display various empty or error states in your application.

## Variants

| Variant           | Description                                              |
|-------------------|---------------------------------------------------------|
| unauthorized      | Access/login verification required                      |
| incomplete        | Data is incomplete                                      |
| notFound          | Not found                                               |
| systemProblem     | System problem                                          |
| badGateway        | Connection lost                                         |
| underMaintenance  | System under maintenance                                |
| gatewayTimeout    | System took too long to respond                         |
| incorrectAccess   | Incorrect way to access                                 |
| custom            | Custom icon, title, description, and action             |

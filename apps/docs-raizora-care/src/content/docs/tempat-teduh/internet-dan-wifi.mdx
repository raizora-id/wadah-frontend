# Internet dan WiFi

## Cara Mengakses WiFi

Kami menyediakan koneksi internet berkecepatan tinggi di seluruh area properti. Berikut panduan untuk mengakses layanan WiFi:

### Setup Awal

1. **Mendapatkan Akses**
   - Anda akan menerima informasi akses WiFi saat check-in
   - Detail akses juga tersedia di buku panduan penghuni
   - Setiap unit mendapatkan akses untuk maksimal 4 perangkat

2. **Informasi Jaringan**
   - Nama jaringan (SSID): KosXXX_Lantai_X (contoh: Kos123_Lantai_2)
   - Untuk area umum: KosXXX_Common
   - Password: Diberikan saat check-in (unik per penghuni)
   - Tipe keamanan: WPA2-Personal

3. **Koneksi Pertama Kali**
   - Pilih jaringan WiFi sesuai lokasi Anda
   - Masukkan password yang diberikan
   - Untuk devices tertentu, Anda mungkin perlu menerima terms & conditions
   - Pada browser akan muncul portal login (captive portal)
   - Masukkan username (nomor kamar) dan password

4. **Registrasi Perangkat**
   - Login ke portal penghuni: wifi.namakos.com
   - Pilih menu "Manajemen Perangkat"
   - Daftar perangkat yang ingin digunakan (max 4)
   - Untuk mengganti perangkat, Anda harus menghapus salah satu dari daftar

### Informasi Teknis

1. **Spesifikasi Koneksi**
   - Kecepatan standar: 10 Mbps per unit (shared)
   - Kecepatan premium: 20-50 Mbps (upgrade berbayar)
   - Teknologi: Fiber Optic
   - Coverage: Seluruh area properti
   - Limit FUP: 300GB/bulan (paket standar)

2. **Titik Akses**
   - Setiap lantai dilengkapi minimal 2 access points
   - Area umum memiliki access points terpisah
   - Signal booster tersedia untuk area tertentu
   - Mesh WiFi system untuk coverage menyeluruh

3. **Fitur Keamanan**
   - Jaringan terenkripsi WPA2
   - Firewall untuk keamanan dasar
   - Pemisahan jaringan antar penghuni
   - Filtering konten dasar

4. **Kebijakan Penggunaan**
   - Tidak diperbolehkan mengunduh konten ilegal
   - Tidak diperbolehkan melakukan aktivitas yang memakan bandwidth berlebihan
   - Fair Usage Policy berlaku untuk semua penghuni
   - Tidak diperbolehkan melakukan aktivitas yang melanggar hukum

## Troubleshooting Koneksi

Mengalami masalah dengan koneksi internet? Coba langkah-langkah berikut sebelum menghubungi bantuan:

### Masalah Umum dan Solusinya

1. **Koneksi Lambat**
   - **Cek jumlah perangkat** yang terkoneksi (max 4 perangkat)
   - **Restart perangkat** Anda (smartphone, laptop)
   - **Hindari penggunaan pada jam sibuk** (19.00-22.00) jika memungkinkan
   - **Tutup aplikasi yang menggunakan bandwidth tinggi** (streaming, download)
   - **Cek jarak dari access point**, pindah lebih dekat jika memungkinkan

2. **Tidak Bisa Terhubung**
   - **Pastikan WiFi aktif** di perangkat Anda
   - **Cek password** yang dimasukkan sudah benar
   - **Coba "Forget Network"** lalu hubungkan kembali
   - **Pastikan perangkat Anda terdaftar** di sistem
   - **Restart router** di unit Anda (jika tersedia)

3. **Koneksi Terputus-putus**
   - **Cek interferensi** dari perangkat elektronik lain
   - **Hindari dinding tebal** atau area dengan penghalang signal
   - **Update driver WiFi** pada laptop/komputer
   - **Coba koneksi di lokasi berbeda** untuk membandingkan

4. **Portal Login Berulang**
   - **Clear cache browser** Anda
   - **Cek cookie settings** (aktifkan cookies)
   - **Pastikan tidak menggunakan VPN** saat login pertama kali
   - **Coba browser berbeda** (Chrome, Firefox, Safari)

### Langkah Reset Koneksi

Jika masalah berlanjut, coba reset koneksi:

1. **Reset Perangkat**
   - Matikan WiFi perangkat selama 30 detik
   - Aktifkan kembali dan hubungkan ke jaringan
   - Jika menggunakan laptop, coba "Network Reset" di pengaturan

2. **Reset Autentikasi**
   - Kunjungi portal.namakos.com/reset
   - Masukkan nomor kamar dan password
   - Pilih "Reset Koneksi"
   - Tunggu 2-3 menit sebelum mencoba terhubung kembali

3. **Hard Reset** (jika tersedia router di unit)
   - Cabut kabel power router selama 1 menit
   - Pasang kembali dan tunggu hingga semua lampu indikator menyala
   - Hubungkan perangkat kembali ke WiFi

### Pelaporan Masalah

Jika langkah-langkah di atas tidak berhasil:

1. **Dokumen yang Diperlukan**
   - Screenshot hasil speed test (dari speedtest.net)
   - Screenshot error message (jika ada)
   - Daftar perangkat yang mengalami masalah
   - Waktu dan durasi masalah

2. **Cara Melapor**
   - Via aplikasi penghuni: menu "Layanan" > "Laporkan Masalah Internet"
   - Via WhatsApp: kirim ke 0812-3456-7890 dengan format "INTERNET - [No.Kamar] - [Deskripsi Masalah]"
   - Via resepsionis: isi formulir keluhan internet

3. **Estimasi Respons**
   - Masalah umum: 2-4 jam kerja
   - Masalah kompleks: 24 jam kerja
   - Masalah jaringan luas: update status via grup WhatsApp/Telegram

## Pertanyaan Umum

**Apakah saya bisa memasang internet sendiri?**
Untuk sebagian besar properti, pemasangan internet pribadi tidak diizinkan karena sudah tersedia layanan bersama. Untuk kasus khusus dengan kebutuhan spesifik, bisa diajukan permohonan ke manajemen.

**Berapa jumlah perangkat yang bisa saya hubungkan?**
Paket standar memungkinkan koneksi hingga 4 perangkat secara bersamaan. Untuk kebutuhan lebih banyak perangkat, tersedia paket upgrade dengan kuota perangkat lebih besar.

**Apakah ada batasan konten yang bisa diakses?**
Ya, kami menerapkan filtering konten dasar untuk memblokir situs ilegal dan konten dewasa. Penggunaan VPN diperbolehkan tetapi mohon tidak digunakan untuk aktivitas ilegal.

**Apa yang terjadi jika melebihi FUP (Fair Usage Policy)?**
Jika Anda melebihi kuota bulanan, kecepatan akan diturunkan menjadi 1-2 Mbps hingga awal bulan berikutnya. Anda dapat membeli paket data tambahan jika diperlukan.

**Apakah jaringan WiFi aman untuk banking dan aktivitas sensitif?**
Ya, jaringan WiFi kami menggunakan enkripsi WPA2 dan isolasi client untuk keamanan dasar. Namun, untuk aktivitas yang sangat sensitif seperti internet banking, kami tetap merekomendasikan penggunaan koneksi data seluler atau VPN tambahan.

**Bagaimana jika saya membutuhkan koneksi stabil untuk meeting online?**
Untuk kebutuhan meeting online atau aktivitas yang membutuhkan koneksi stabil, kami menyarankan upgrade ke paket Plus atau Pro. Area common dengan WiFi dedicated juga tersedia untuk meeting penting.
